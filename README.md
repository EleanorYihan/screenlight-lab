# ScreenLight Lab

屏幕灯产品信息提取与竞品分析工作台。它把品牌、型号、产品定义、用户画像、价格、规格、适配屏幕和技术拆解统一到一个可检索、可对比、可导出的研究界面中。

## 功能

- 从淘宝、天猫、京东等公开商品页读取标题、品牌线索、价格和商品编号
- 上传商品详情长图或参数截图，在浏览器内进行中英文 OCR
- 自动把识别文字映射为型号、尺寸、色温、显色度、功率、输入、角度、适配屏幕与技术方案
- 手动补全并核验研究字段
- 保存到当前浏览器，无需登录即可使用
- 搜索产品，查看产品详情，选择 2–4 款横向对比
- 导出带中文表头的 CSV
- 响应式桌面与移动端界面

示例产品为虚构演示数据，不代表真实商品规格。网页提取结果应回到官网、说明书或实测来源人工核验。

## 本地运行

需要 Node.js 22.13 或更高版本。

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
```

## 技术说明

项目基于 React、Next.js 兼容接口、vinext 与 Cloudflare Workers。`/api/extract` 在服务端读取公开网页并优先解析 JSON-LD Product 信息；长图识别通过按需加载的 Tesseract.js 中文模型在本地浏览器执行。线上部署需要支持服务端运行，不能只作为纯静态 GitHub Pages 站点托管。

淘宝、天猫与京东可能要求登录、滑块验证或限制自动访问。因此链接提取用于尽量补全基础信息，详情长图 OCR 是更稳定的参数入口，最终数据仍应回到商品页或说明书核验。
